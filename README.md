### Описание:
Сайт для продажи и покупки авто. Размещение, редактирование и удаление обьявлений.
Добавить может только авторизированный пользователь. Есть возможность фильтрации по параметрам.

### Используемые технологии:
- Servlet
- Hibernate
- Jsp
- JSTL
- HTML
- CSS
- JavaScript
